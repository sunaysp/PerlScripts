This tool creates tag from branch and updates tagURL in exchange/.*/pom.xml, build.bat, build.properties 

Open config.properties file. Set values. 
1. tagName: TagName to be created. If tagName is already exists, it will not create. 
2. comment: SVN Log comment for tag creation. Give [artifactId] followed by particular log message. 
3. antpath: If you want to change ant.lib.dir value in AutomatedBuildScripts/build.properties, Set the new value here.

If you need to create tag from trunk or from another branch, change srcURL Value in createtag.bat file to new branch or trunk URL.
